Installation

1) Ensure that the Android SDK is installed on your machine, specifically the Android Debug Bridge ("adb").

2) Connect your Android (Gingerbread version) to your machine via USB.

3) Download the DroidWatch-Gingerbread-2.3.6.apk file.

4) Type the following command: adb install DroidWatch-Gingerbread-2.3.6.apk

5) Reboot the phone.
